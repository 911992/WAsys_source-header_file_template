In Eclipse
Window menu -> Preferences -> C/C++ -> Code Style -> Code Template
Now in right setting panel, open "Files" root tree item, then "C source file" and "C header file"

Import "c_source_file_template.xml" and "c_header_file_template.xml" for "Default C source file template" and "Default C header file template"

Try to generate the header files first when coding

Happy programming
